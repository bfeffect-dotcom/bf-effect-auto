# BF Effect News

Автоматический Telegram-бот для публикации важных мировых экономических, корпоративных и геополитических новостей в канал **@bf_effect_news**.

---

## О проекте

BF Effect News автоматически собирает новости из ведущих финансовых источников, отбирает только значимые события и публикует их в Telegram в кратком и удобном формате.

Основная цель проекта — предоставить подписчикам быстрый доступ к событиям, которые могут влиять на мировую экономику, финансовые рынки, сырьевые товары, технологический сектор и международную политику.

---

## Основные возможности

* Автоматический сбор новостей из нескольких источников
* Фильтрация по редакционным правилам
* Исключение аналитических и рекламных материалов
* Автоматический перевод на русский язык
* Защита от публикации дубликатов
* Полностью автоматическая работа через GitHub Actions
* Публикация новостей напрямую в Telegram

---

## Источники данных

В настоящий момент бот использует следующие источники:

* CNBC

## Какие материалы не публикуются

Бот автоматически исключает:

* инвестиционные рекомендации;
* ETF и ETF-обзоры;
* подборки акций;
* мнения аналитиков;
* прогнозы рынка;
* Weekly Outlook;
* Week Ahead;
* What To Watch;
* контент Motley Fool;
* статьи без конкретного новостного события;
* рекламные материалы.

## Как работает система

Во время каждого запуска бот выполняет следующие действия:

1. Загружает новости из RSS-источников.
2. Очищает данные от HTML и служебной информации.
3. Проверяет новости по редакционным правилам.
4. Отбрасывает нерелевантный контент.
5. Переводит материалы на русский язык.
6. Формирует публикацию.
7. Отправляет её в Telegram.
8. Сохраняет идентификатор новости для предотвращения повторной публикации.

## Настройка

Для работы проекта необходимо создать секрет GitHub Actions:

```text
TELEGRAM_BOT_TOKEN
```

Добавить его можно в:

```text
Settings → Secrets and variables → Actions
```

Значение секрета — токен Telegram-бота, полученный через BotFather.

---

## Ручной запуск

Для запуска бота вручную:

```text
GitHub → Actions → Market News Bot → Run workflow
```

---

## Автоматическое расписание

Бот запускается автоматически каждые 4 часа через GitHub Actions.

Текущая конфигурация:

```yaml
cron: '0 */4 * * *'
```

## Отказ от ответственности

Информация, публикуемая ботом, предоставляется исключительно в информационных целях.

Проект не предоставляет инвестиционные рекомендации, финансовые консультации или торговые сигналы.

Все материалы основаны на открытых источниках и публикуются в автоматическом режиме.
