# BF Effect News

Автоматический Telegram-бот для публикации важных мировых экономических, корпоративных и геополитических новостей в канал **@bf_effect_news**.

## О проекте

BF Effect News автоматически отслеживает новости из открытых финансовых источников, отбирает только значимые события и публикует их в Telegram в кратком и понятном формате.

Цель проекта — быстро показывать события, способные повлиять на мировую экономику, финансовые рынки, сырьевые товары, технологический сектор и международную политику.

## Основные возможности

* Автоматический сбор новостей из нескольких источников.
* Фильтрация по важным экономическим и рыночным событиям.
* Исключение аналитических статей, инвестиционных рекомендаций и низкокачественного контента.
* Автоматический перевод новостей на русский язык.
* Защита от публикации дубликатов.
* Полностью автоматическая работа через GitHub Actions.

## Как работает бот

GitHub Actions автоматически запускает `news_bot.py` по расписанию.

Во время запуска бот:

1. Загружает новости из RSS-лент.
2. Очищает текст от HTML и служебных данных.
3. Фильтрует публикации по редакционным правилам.
4. Переводит заголовок и описание на русский язык.
5. Формирует публикацию.
6. Отправляет её в Telegram.
7. Сохраняет идентификатор новости для защиты от дублей.

## Формат публикации

```text
Заголовок

Краткое описание события.

Источник: CNBC
```

## Настройка

Открой:

```text
Settings → Secrets and variables → Actions
```

Добавь секрет:

```text
TELEGRAM_BOT_TOKEN
```

Значение — токен Telegram-бота, полученный через BotFather.

## Ручной запуск

```text
Actions → Market News Bot → Run workflow
```

## Расписание

Текущая настройка GitHub Actions:

```yaml
cron: '0 */4 * * *'
```

Бот запускается каждые 4 часа.
